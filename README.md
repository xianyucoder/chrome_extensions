# chrome_extensions
谷歌插件开发

videocc-download 为插件，chrome安装，请进入开发者模式加载。

获取到key 后，可以使用  https://tools.heisir.cn/HLSDownload/ M3U8-Downloader下载m3u8

目前只针对指定网站获取，而我知道的网站有限，如果您的网站安装插件后没有反应，请留言

当前支持的网站：

www.sciconf.cn

www.weplanning.cn

使用方法：

1.安装插件

2.进入对应的视频页面

3.在显示的窗口内复制key以及m3u8地址

4.使用M3U8-Downloader下载
